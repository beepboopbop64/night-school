# Review packet: dotprod, round-02

How to review, exactly as a student would:

1. `cd site` (from the lessons repo root)
2. `npm install` (first time only)
3. `npm run preview` (serves the built site from `dist/`)
4. Open `http://localhost:4173/sessions/dotprod`

The video is embedded on the page (chaptered per beat, captions on) and
also on disk at `sessions/dotprod/media/session.mp4`.
The written doc: `sessions/dotprod/doc/lesson.md`.
The extensions menu: `sessions/dotprod/extensions.yaml`.
The notebook: `sessions/dotprod/notebook/lesson.ipynb`.

Write all notes in `../notes.md` under the pre-seeded anchors, then run
`harness round dotprod`. Open flags: `../flags.md`. The delivery
contract: `../../../manifest.yaml`. The screening verdicts:
`../screening-report.md`.
