# Review packet: toy, round-01

How to view the session as a student would:

1. `cd site` (from the lessons repo root)
2. `npm install` (first time only)
3. `npm run preview` (serves the built site from `dist/`)
4. Open `http://localhost:4173/sessions/toy`

The rendered video is not hosted (open decision D-O1). Play it from disk:
`sessions/toy/media/session.mp4`.
